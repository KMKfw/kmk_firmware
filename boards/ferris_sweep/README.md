# Ferris Sweep

The Ferris Sweep is a split, 34 key keyboard designed by David Philip Barr - its repo is here:
https://github.com/davidphilipbarr/Sweep

It's based in turn on the Ferris by Philip Chevalier. The default keymap used as reference is here:
https://github.com/qmk/qmk_firmware/tree/master/keyboards/ferris/keymaps/default
